# cycosta.github.io

Codebase: React + TypeScript + Vite

## Available scripts

**Run in development mode**

```zsh
  npm run dev
```

**Run build**

```zsh
  npm run build
```

**Run Lint**

```zsh
  npm run lint
```

**Format files using Prettier**

```zsh
  npm run prettier
```

**Run unit tests**

```zsh
  npm run test
```
